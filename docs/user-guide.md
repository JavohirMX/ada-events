## Getting Started

### Accessing the Site

Open your browser and navigate to:
```
https://your-domain.com
```

Or use the IP address if domain is not set up:
```
http://your-droplet-ip
```

### First Time Users

1. Click **Sign Up** in the navigation
2. Enter your email and password
3. Complete your profile (optional)
4. Start browsing events!

## Creating Events

### Who Can Create?

Any logged-in member can create events!

### Step-by-Step

1. Click **Create Event** in navigation (requires login)
2. Fill in event details:

| Field | Required | Description |
|-------|----------|-------------|
| Title | Yes | Event name |
| Category | No | Event type |
| Description | Yes | What to expect |
| Date | Yes | When it happens |
| Time | No | Start time |
| Location | Yes | Where (be specific!) |
| Location URL | No | Google Maps link |
| Cover Image | No | Event banner |
| Max Attendees | No | Leave blank for unlimited |
| Public Attendees | Yes | Can others see who's coming? |
| WhatsApp Link | No | Group for event chat |

3. Click **Create Event**
4. Your event is now live!

### Tips for Good Events

- Add a clear description of what to expect
- Include meeting point details
- Add WhatsApp group for coordination
- Set a reasonable max attendees

## RSVPing to Events

### Quick RSVP (One-Click)

From event list, click **RSVP** button on any event card.

### Full RSVP

1. Click on event to open detail page
2. Choose response:
   - **Going** - You plan to attend
   - **Maybe** - You might attend
   - **Can't Go** - You won't attend
3. Your response is saved immediately

If event is full and you choose **Going**, you are automatically added to waitlist.

### Changing Your RSVP

1. Go to event detail page
2. Click a different response button
3. Your RSVP is updated

If a confirmed attendee leaves, first person in waitlist is promoted automatically.

### Viewing Attendees

- If event has **public** attendee list, anyone can see who's coming
- If **private**, only the event creator can see the list

## Managing Your Events

### As an Event Creator

You can edit events you created:

1. Go to your event's detail page
2. Click the **edit icon** (pencil) in the top right
3. Update details
4. Click **Save Changes**

### Adding Gallery After Event

After your event has passed:

1. Edit your event
2. **Gallery Link** field becomes available
3. Add link to:
   - Google Photos album
   - Google Drive folder
   - Dropbox folder
   - Any photo sharing service
4. Save changes

The gallery link appears on the event detail page!

Server also enforces this rule, so gallery link cannot be set before event date.

### Attachments

When creating/editing events you can upload attachments:
- Allowed: `jpg`, `jpeg`, `png`, `gif`, `webp`, `pdf`, `doc`, `docx`
- Max size: 10MB per file
- Max files: 5 per event

## FAQ

### Q: Do I need to verify my email?

**A:** No! For this community platform, we skip email verification. Just sign up with any email.

### Q: Can anyone create events?

**A:** Yes! Any logged-in member can create events.

### Q: Can I change my RSVP after submitting?

**A:** Yes, go to the event and click a different response.

### Q: What happens if an event reaches max attendees?

**A:** New "Going" requests are added to waitlist automatically. FIFO promotion happens when a confirmed attendee leaves.

### Q: When can I add event photos?

**A:** After the event date has passed, the gallery link field becomes editable.

### Q: How do I join the WhatsApp group?

**A:** Click the WhatsApp link on the event detail page.

### Q: Can I hide my attendance?

**A:** Yes! Event creators can choose private attendee lists.

### Q: Who can see my profile?

**A:** Any logged-in member of the community can view profiles.
